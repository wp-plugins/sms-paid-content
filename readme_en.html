How the module works:


This plugin allows you to provide a paid access to any content on your website.
In reply message user receives a short password (keyword)text
which enables him to access any content you hide with this sms:key.
You decide how many times and how long this password is valid for.

Example:


Setup process:

  • Copy "smscoin_key" folder from the archive to the folder: wp-content/plugins


  • Go to Admin panle of your "WordPress", in Plugins section,
    You will find "SMSCOIN Key", click to activate.


  • Go to "Options" section, choose the tab "SmsCoin - sms:key",
    and follow the instructions for plugin configuration.


    • Example:

    Installation process is completed!


    How it works:


  • To article text add those tags you have specified in plugin settings.
    Example: [smscoin_key] hidden text [/smscoin_key].

    • Uninstallation:

  • Go to Admin panle of your "WordPress", in Plugins section,
    Find and deactivate "SMSCOIN Key".

  • Delete the folder wp-content/plugins/smscoin_key


    • Module was successfully uninstalled.
